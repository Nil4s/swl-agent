CORE ELECTRONICS (ARDUINO BASED)
--------------------------------------------------------------------------------
[1x] Arduino Nano V3 or Clone
     - ATmega328P microcontroller (16MHz)
     - 5V logic, 3.3V regulator onboard
     - USB programming interface
     - Cost: $3-8 (clone), $25 (official)
     - Recommended: CH340 USB chip version (cheaper, works great)
     - Source: AliExpress, Amazon, eBay
     
[1x] MPU6050 GY-521 IMU Module
     - 6-axis gyroscope + accelerometer
     - I2C interface (easy connection)
     - 3.3V or 5V compatible
     - Cost: $1-3
     - CRITICAL: This is your stabilization sensor
     - Source: AliExpress, Amazon
     
[1x] HC-05/HC-06 Bluetooth Module OR NRF24L01+ Radio
     - Bluetooth: $3-5 (40m range, simple pairing)
     - NRF24: $1-3 (100m range, better latency)
     - Recommended: NRF24L01+ with PA+LNA (1km range, $5)
     - Source: AliExpress, Amazon

[1x] PCA9685 16-Channel PWM Servo Driver
     - I2C controlled servo/ESC driver
     - Drives 3x servos + 5x ESCs simultaneously
     - Onboard voltage regulation
     - Cost: $2-5
     - Alternative: Use Arduino PWM pins directly (limited to 6 outputs)
     - Source: AliExpress, Amazon

POWER MANAGEMENT
--------------------------------------------------------------------------------
[1x] LM2596 Buck Converter (or equivalent)
     - Input: 7.4-14.8V (2S-4S LiPo)
     - Output: 5V @ 3A for Arduino + servos
     - Adjustable potentiometer for voltage trim
     - Cost: $1-2
     - Alternative: L7805 linear regulator (cheap but hot)
     - Source: AliExpress, Amazon

[1x] XT30 or XT60 Battery Connector
     - XT30: Up to 30A (good for this build)
     - XT60: Up to 60A (overkill but robust)
     - Cost: $0.50-1 per connector
     - Get male + female pair
     - Source: Hobby stores, Amazon

[4x] 2S LiPo Batteries (7.4V)
     - Capacity: 500-1000mAh each
     - C-rating: 25C minimum
     - Configuration: 2S2P (7.4V, 2000mAh total)
     - Alternative: Single 3S 1500mAh battery (11.1V)
     - Cost: $8-12 per battery
     - Source: Hobbyking, Banggood, local hobby shop
     
[1x] LiPo Balance Charger
     - 2S-4S capable
     - IMAX B6 clone recommended ($12-20)
     - CRITICAL: Never use USB charger for LiPo!
     - Cost: $12-25
     - Source: Amazon, Hobbyking

COST SAVINGS TIPS
--------------------------------------------------------------------------------
1. **Buy from AliExpress** - 50-70% cheaper than Amazon
   - Downside: 2-4 week shipping
   - Upside: Same products, way cheaper

2. **Get Arduino clones** - $3 vs $25 for official
   - Functionality identical
   - CH340 USB chip works perfectly

3. **Buy servo packs** - 5x SG90 for $8 vs $2 each
   - Use extras for other projects

4. **Skip optional upgrades initially**
   - OLED, buzzer, sensors can be added later
   - Focus on core flight first

5. **Reuse old charger if possible**
   - Any 2S-4S LiPo charger works
   - Borrow from friend to test

6. **3D print everything**
   - Avoid carbon fiber frame ($30 savings)
   - PLA/PETG works fine for testing

7. **Direct solder connections**
   - Skip JST connectors initially (save $5)
   - Add connectors later for convenience

8. **Single 3S battery instead of 4x 2S**
   - 3S 1500mAh: $12-15
   - Easier power distribution
   - Less weight

ARDUINO FLIGHT CODE
--------------------------------------------------------------------------------

- MultiWii 2.4: Free, open source, Arduino compatible
- Supports MPU6050, servos, ESCs
- Basic stabilization (no GPS/fancy features)

- Full control over flight logic
- Steeper learning curve

WIRING DIAGRAM (SIMPLIFIED)
--------------------------------------------------------------------------------
Arduino Nano Connections:
  5V  ‚Üí Buck converter 5V output
  GND ‚Üí Common ground (battery negative)
  A4  ‚Üí MPU6050 SDA (I2C data)
  A5  ‚Üí MPU6050 SCL (I2C clock)
  
  D9  ‚Üí PCA9685 SDA (I2C)
  D10 ‚Üí PCA9685 SCL (I2C)
  
  D2  ‚Üí NRF24 CE
  D3  ‚Üí NRF24 CSN
  D11 ‚Üí NRF24 MOSI
  D12 ‚Üí NRF24 MISO
  D13 ‚Üí NRF24 SCK

PCA9685 Outputs:
  PWM 0-2  ‚Üí 3x Gimbal Servos
  PWM 3    ‚Üí Main Motor ESC
  PWM 4-7  ‚Üí 4x Peripheral Motor ESCs

Power Distribution:
  Battery + ‚Üí Buck converter IN+
  Battery - ‚Üí Common GND
  Buck 5V   ‚Üí Arduino, PCA9685, Servos
  Battery   ‚Üí ESCs directly (7.4V or 11.1V)

IMPORTANT NOTES:
- MPU6050 must be mounted flat, centered on frame
- Use 1000¬µF capacitor on battery input (prevent voltage spikes)
- Separate power for motors (direct from battery) and Arduino (via buck)
- All grounds must be connected (common ground)

SAFETY CHECKLIST
--------------------------------------------------------------------------------
‚ñ° All solder joints insulated with heat shrink
‚ñ° No exposed wire strands (shorts = fire)
‚ñ° Battery connector polarity verified (RED = +, BLACK = -)
‚ñ° Buck converter output voltage measured (must be 5.0-5.2V)
‚ñ° LiPo voltage checked before every flight (>3.5V per cell)
‚ñ° Failsafe programmed (motors off if signal lost)
‚ñ° Props removed during all bench testing
‚ñ° Fire extinguisher nearby (LiPo fires are intense)
