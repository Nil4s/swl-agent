GIMBAL IMPELLER DRONE - BILL OF MATERIALS

Design Specs: 90mm main impeller, 4x 50mm peripheral bladeless impellers
Total wingspan: ~520mm, Estimated weight: 350-450g without batteries

MOTORS & PROPULSION
--------------------------------------------------------------------------------
[1x] Main Impeller Motor
     - Brushless motor 2204-2206 size, 2300-2600 KV
     - Shaft: 5mm, M3 mounting (16x16mm or 19x19mm pattern)
     - Recommended: Emax RS2205 2300KV or equivalent
     - Power: 15-20A @ 11.1V (3S LiPo)
     
[4x] Peripheral Impeller Motors
     - Micro brushless 1105-1306 size, 4000-5000 KV
     - Compact, lightweight (<15g each)
     - Recommended: Emax RS1106 4500KV or GEPRC 1204 5000KV
     - Power: 4-8A each @ 11.1V

[5x] Motor ESCs (Electronic Speed Controllers)
     - Main: 20A+ BLHeli_S or BLHeli32
     - Peripheral: 10-12A micro ESC (4x)
     - Alternative: 4-in-1 ESC (20A x4) + separate 20A for main
     - Recommended: 30A 4-in-1 BLHeli32 ESC

SERVOS & ACTUATORS
--------------------------------------------------------------------------------
[3x] Gimbal Control Servos (for main impeller tilt)
     - MG90S Metal Gear Digital Servo
     - Torque: 1.8 kg·cm @ 4.8V, 2.2 kg·cm @ 6V
     - Speed: 0.1 sec/60° @ 4.8V
     - Weight: 13.4g each
     - Angle: 180° (±90°)
     
[4x] Peripheral Impeller Tilt Servos (optional - if using vectored thrust)
     - SG90 or MG90S micro servo
     - Weight: 9g each
     - Torque: 1.8 kg·cm

AUTONOMOUS FLIGHT CAPABILITIES (Software Features)
--------------------------------------------------------------------------------
With GPS + Sensors + AI Camera, enables:

1. **Return to Home (RTH)**
   - Automatic return on low battery or signal loss
   - GPS waypoint navigation
   
2. **Orbit Mode**
   - Circle around a GPS point or visual target
   - AI camera tracks subject automatically
   
3. **Follow Me**
   - GPS tracker on user (phone or dedicated)
   - AI vision tracks user movement
   - Maintains safe distance and altitude
   
4. **Waypoint Missions**
   - Pre-programmed flight paths
   - Survey, inspection, delivery routes
   - INAV or ArduPilot firmware
   
5. **Precision Landing**
   - Optical flow + ToF sensor
   - AprilTag/ArUco marker recognition
   - Lands on moving platform (car roof)
   
6. **Obstacle Avoidance**
   - Ultrasonic + optical flow fusion
   - AI vision detects obstacles
   - Auto-braking and path re-routing
   
7. **Gesture Control**
   - AI camera recognizes hand signals:
     * Takeoff (palm up)
     * Land (palm down)
     * Follow (point)
     - Selfie (peace sign)
   
8. **Object Tracking**
   - AI locks onto person, car, ball, etc.
   - Smooth gimbal follows target
   - Auto-zoom (if using HD camera with digital zoom)

NOTES & CONSIDERATIONS
================================================================================

1. BATTERY CLARIFICATION NEEDED:
   - "5V LiPo bacon strips" are unusual. Standard options:
     * 1S 3.7V 500-1000mAh pouch cells (need voltage boost to 5V)
     * 2S 7.4V 500-1000mAh (still need regulation)
     * Use standard 4S 14.8V drone battery instead?

2. GIMBAL COMPLEXITY:
   - 3-servo gimbal control is mechanically complex
   - Consider 2-servo system (pitch + roll only)
   - Yaw from differential peripheral motor thrust

3. FLIGHT CONTROLLER CONFIGURATION:
   - Non-standard design requires custom mixer in Betaflight
   - May need INAV for better support of unconventional frames
   - Extensive tuning required

4. TESTING RECOMMENDATIONS:
   - Test main impeller thrust separately before integration
   - Calibrate servo travel limits carefully (avoid binding)
   - Start with conservative gimbal angles (±15°) before full ±30°
   - Test each peripheral motor independently

5. SAFETY:
   - Always use LiPo safety bag for charging
   - Never leave batteries unattended while charging
   - Test in open area away from people/property
   - Use propeller guards during initial testing

6. ALTERNATIVE DESIGNS TO CONSIDER:
   - Eliminate peripheral motor vectoring (fixed angle)
   - Use 2-axis gimbal only (simpler)
   - Conventional propellers instead of bladeless (more efficient)
